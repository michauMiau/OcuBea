OcuBea - IP Camera Streamer

Status: Streaming via /video endpoint

Available Endpoints (IP Webcam Compatible)

  • /video — MJPEG stream (auto-refreshing)
  • /shot.jpg — Single frame snapshot
  • /focus / /nofocus — Focus control
  • /settings/night_vision?set=on|off
  • /settings/ffc?set=on|off (POST)
  • /ptz?zoom=N (POST, N=10-50)
  • /settings/quality?set=90 (POST)
